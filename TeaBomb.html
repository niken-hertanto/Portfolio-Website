Tea Bomb
Level Desginer, Artist
November 2017 - November 2017

About the Game:

Tea Bomb is a 2-player 2D fighting game where players toss bombs at each other. The game is made to include the narrative twist "... and then they had tea." To show this twist, at any time during the fight players can walk into the tea house. Here, all 4 characters sit around a table and have tea. This game was made with Unity and C# in 2 weeks.

Contributions:

  • Programmed the character select menu using C#. Designed and created the artwork used to portray the narrative in the menu using Photoshop and Paint Tool Sai.
