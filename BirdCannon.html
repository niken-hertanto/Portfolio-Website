Bird Cannon
Programmer, Artist
November 2017 - November 2017

About the Game:

Bird Cannon is a 2D game where the player plays as a bird whom anticipates being shot out from a cannon. The player aims at a button at the other end of the screen, where each time the button is pressed the fuse of the cannon gets shorter and shorter. This game is made with Unity and C#.

Contributions:

  • Hand-painted all the artwork for the game using Photoshop and Paint Tool Sai.
  • Designed and programmed the gacha wheel with C#.
  • Designed and created the artwork for all the explosions.
